# Frozen / Historical Reports

This folder contains documentation that accurately reflects the **previous state** of the codebase and project progress (as it existed on `main` before the recent intuitive shell, Discover/Xplore terminology, Opportunity target refactoring, and related changes).

These files were moved here (instead of leaving them at the `reports/` root with heavy edits) so that:

- The active reports at the root (`batch-2-engineering-blueprints.md`, `intuitive-shell-plan.md`, `single-pr-intuitive-product.md`) can be kept up-to-date and relevant to current work.
- Historical documents are not polluted with modern terminology or structural changes that would make them inaccurate records of "what we thought at the time."

Each file in this folder starts with a note indicating it is frozen.

**Do not edit these files** unless you are intentionally backporting a correction to the historical record.

See the three reports in the parent `reports/` directory for documents that were intentionally updated as part of the recent changes.
